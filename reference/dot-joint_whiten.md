# Joint whitening on stacked X/Y prototypes to place them in common coords Returns whitened fits for X and Y plus the whitening transform.

Joint whitening on stacked X/Y prototypes to place them in common coords
Returns whitened fits for X and Y plus the whitening transform.

## Usage

``` r
.joint_whiten(X_fit, Y_fit, do_shrink = TRUE, center = TRUE, eps = 1e-07)
```

## Value

A list with whitened data matrices.
