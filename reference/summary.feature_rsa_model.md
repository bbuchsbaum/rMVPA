# Summary Method for Feature RSA Model

Summary Method for Feature RSA Model

## Usage

``` r
# S3 method for class 'feature_rsa_model'
summary(object, ...)
```

## Arguments

- object:

  The feature RSA model

- ...:

  Additional args

## Value

A list of summary statistics for the feature RSA model (printed as side
effect).

if (FALSE) { # \dontrun{
