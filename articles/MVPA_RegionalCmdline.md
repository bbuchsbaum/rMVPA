# MVPA Regional Analysis Tutorial

## Superseded

This vignette has been superseded by
[`vignette("CommandLine")`](http://bbuchsbaum.github.io/rMVPA/articles/CommandLine.md),
which now documents the supported command-line interface for both:

- `rmvpa-searchlight`
- `rmvpa-regional`

Use the unified vignette for:

- installation and `PATH` setup
- wrapper discovery with `--help`, `--version`, `--list-models`, and
  `--dry-run`
- shared workflow flags and config precedence
- searchlight-specific options and examples
- regional-specific options and examples, including ROI masks and
  pooling

If you arrived here from an older link, switch to
[`vignette("CommandLine")`](http://bbuchsbaum.github.io/rMVPA/articles/CommandLine.md)
for the current interface.
